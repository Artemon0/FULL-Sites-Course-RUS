# Проект 1: Личное Портфолио

## Цель проекта
Создать одностраничное портфолио с использованием семантического HTML.

## Требования

### Обязательные элементы:
- [ ] Шапка с навигацией
- [ ] Секция "О себе" с фото
- [ ] Секция "Навыки" со списком
- [ ] Секция "Проекты" с карточками проектов
- [ ] Контактная форма
- [ ] Подвал с копирайтом

### Технические требования:
- Использовать семантические теги HTML5
- Все изображения должны иметь alt атрибуты
- Форма должна иметь валидацию
- Навигация должна работать через якорные ссылки

## Шаги выполнения

1. Создайте базовую структуру HTML
2. Добавьте навигацию
3. Создайте секцию героя с приветствием
4. Добавьте секцию "О себе"
5. Создайте список навыков
6. Добавьте карточки проектов
7. Создайте контактную форму
8. Добавьте подвал

## Дополнительные задания

- Добавьте таблицу с опытом работы
- Встройте видео-презентацию
- Добавьте ссылки на социальные сети
- Создайте отдельную страницу для каждого проекта

## Результат

После завершения у вас будет полноценное HTML портфолио, готовое для стилизации в следующем модуле!
