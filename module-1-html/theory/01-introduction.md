# Модуль 1: Основы HTML

## Что такое HTML?

HTML (HyperText Markup Language) - это язык разметки, который используется для создания структуры веб-страниц.

## Базовая структура HTML документа

```html
<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Название страницы</title>
</head>
<body>
    <!-- Содержимое страницы -->
</body>
</html>
```

## Основные элементы

### Заголовки
```html
<h1>Главный заголовок</h1>
<h2>Подзаголовок</h2>
<h3>Заголовок третьего уровня</h3>
<!-- h4, h5, h6 -->
```

### Параграфы и текст
```html
<p>Это параграф текста.</p>
<strong>Жирный текст</strong>
<em>Курсивный текст</em>
<br> <!-- Перенос строки -->
```

### Ссылки
```html
<a href="https://example.com">Текст ссылки</a>
<a href="page.html">Внутренняя ссылка</a>
<a href="#section">Якорная ссылка</a>
```

### Изображения
```html
<img src="image.jpg" alt="Описание изображения">
```

### Списки
```html
<!-- Маркированный список -->
<ul>
    <li>Элемент 1</li>
    <li>Элемент 2</li>
</ul>

<!-- Нумерованный список -->
<ol>
    <li>Первый</li>
    <li>Второй</li>
</ol>
```

## Семантические элементы HTML5

```html
<header>Шапка сайта</header>
<nav>Навигация</nav>
<main>Основной контент</main>
<article>Статья</article>
<section>Секция</section>
<aside>Боковая панель</aside>
<footer>Подвал сайта</footer>
```
