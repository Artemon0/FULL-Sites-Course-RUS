# Проект 3: Интерактивная галерея

## Цель проекта
Создать красивую интерактивную галерею изображений с различными эффектами и фильтрами.

## Требования

### Обязательные элементы:
- [ ] Сетка изображений (Masonry layout)
- [ ] Фильтры по категориям
- [ ] Модальное окно для просмотра
- [ ] Навигация между изображениями
- [ ] Плавные анимации
- [ ] Эффекты при наведении
- [ ] Адаптивный дизайн

### Технические требования:
- Использовать CSS Grid или Flexbox
- Анимации и переходы
- Transform для эффектов
- Модальное окно на чистом CSS
- Адаптивность для всех устройств

## Примеры эффектов

### Эффект при наведении
```css
.gallery-item {
    position: relative;
    overflow: hidden;
    transition: transform 0.3s ease;
}

.gallery-item:hover {
    transform: scale(1.05);
    box-shadow: 0 10px 30px rgba(0,0,0,0.3);
}

.gallery-item img {
    transition: transform 0.5s ease;
}

.gallery-item:hover img {
    transform: scale(1.1);
}
```

### Анимация появления
```css
@keyframes fadeInUp {
    from {
        opacity: 0;
        transform: translateY(30px);
    }
    to {
        opacity: 1;
        transform: translateY(0);
    }
}

.gallery-item {
    animation: fadeInUp 0.6s ease-out;
}
```

### Flip эффект
```css
.card {
    perspective: 1000px;
}

.card-inner {
    position: relative;
    width: 100%;
    height: 100%;
    transition: transform 0.6s;
    transform-style: preserve-3d;
}

.card:hover .card-inner {
    transform: rotateY(180deg);
}

.card-front,
.card-back {
    position: absolute;
    width: 100%;
    height: 100%;
    backface-visibility: hidden;
}

.card-back {
    transform: rotateY(180deg);
}
```
