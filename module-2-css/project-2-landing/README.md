# Проект 2: Лендинг страница

## Цель проекта
Создать современную адаптивную лендинг страницу с использованием Flexbox, Grid и адаптивного дизайна.

## Требования

### Обязательные секции:
- [ ] Героическая секция с фоновым изображением
- [ ] Секция "Преимущества" (3-4 карточки)
- [ ] Секция "О продукте" с изображением
- [ ] Секция "Отзывы" (слайдер или карточки)
- [ ] Секция "Цены" (3 тарифа)
- [ ] Контактная форма
- [ ] Футер с ссылками

### Технические требования:
- Адаптивный дизайн (мобильные, планшеты, десктоп)
- Использование Flexbox и Grid
- Плавные переходы на кнопках и ссылках
- Оптимизированные изображения
- Валидный HTML и CSS

### Дизайн требования:
- Единая цветовая схема
- Читаемая типографика
- Достаточные отступы и пространство
- Визуальная иерархия

## Шаги выполнения

1. Создайте HTML структуру всех секций
2. Настройте базовые стили (цвета, шрифты)
3. Стилизуйте героическую секцию
4. Создайте сетку карточек преимуществ
5. Оформите секцию о продукте
6. Добавьте карточки отзывов
7. Создайте таблицу цен
8. Стилизуйте форму
9. Оформите футер
10. Добавьте адаптивность
11. Добавьте анимации и переходы

## Дополнительные задания

- Добавьте липкую навигацию
- Создайте плавную прокрутку к секциям
- Добавьте модальное окно
- Создайте анимацию появления элементов при скролле
- Добавьте темную тему

## Результат

Полностью адаптивная лендинг страница, готовая для использования в реальных проектах!
