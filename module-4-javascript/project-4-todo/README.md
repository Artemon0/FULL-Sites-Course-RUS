# Проект 4: Todo приложение и Калькулятор

## Часть 1: Todo приложение

### Цель
Создать полнофункциональное приложение для управления задачами.

### Требования

**Функционал:**
- [ ] Добавление новых задач
- [ ] Отметка задач как выполненных
- [ ] Удаление задач
- [ ] Редактирование задач
- [ ] Фильтрация (все/активные/выполненные)
- [ ] Счётчик активных задач
- [ ] Очистка выполненных задач
- [ ] Сохранение в LocalStorage

**Технические требования:**
- Использовать DOM манипуляции
- Обработка событий
- Работа с LocalStorage
- Валидация ввода

### Структура

```
project-4-todo/
├── index.html
├── style.css
└── script.js
```

### Шаги выполнения

1. Создать HTML разметку
2. Стилизовать интерфейс
3. Реализовать добавление задач
4. Добавить отметку выполнения
5. Реализовать удаление
6. Добавить редактирование
7. Создать фильтры
8. Сохранять в LocalStorage
9. Добавить анимации

### Дополнительные задания

- Добавить приоритеты задач
- Реализовать категории
- Добавить дедлайны
- Создать поиск по задачам
- Добавить drag & drop для сортировки

## Часть 2: Калькулятор

### Цель
Создать функциональный калькулятор с красивым интерфейсом.

### Требования

**Функционал:**
- [ ] Базовые операции (+, -, *, /)
- [ ] Очистка (C и CE)
- [ ] Проценты
- [ ] Смена знака (+/-)
- [ ] Десятичные числа
- [ ] История вычислений
- [ ] Клавиатурный ввод

**Технические требования:**
- Обработка событий клавиатуры
- Валидация ввода
- Обработка ошибок (деление на 0)
- Сохранение истории

### Шаги выполнения

1. Создать интерфейс калькулятора
2. Стилизовать кнопки
3. Реализовать ввод чисел
4. Добавить операции
5. Реализовать вычисления
6. Добавить очистку
7. Реализовать клавиатурный ввод
8. Добавить историю
9. Обработать ошибки

### Дополнительные задания

- Добавить научные функции (sin, cos, sqrt)
- Реализовать память (M+, M-, MR, MC)
- Добавить темы оформления
- Создать режим программиста (двоичная система)
- Добавить конвертер единиц

## Результат

После завершения у вас будет два полноценных приложения, демонстрирующих работу с DOM, событиями и LocalStorage!
